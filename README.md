Yield Sift Science
===

A promise-wrapped helper lib for yielding Sift Science API calls

### Installation

Using [npm](https://npmjs.org):

```bash
$ npm install otothea/yield-siftscience
```

### Usage

```js
var siftscience = require('yield-siftscience')('YOUR_SIFT_SCIENCE_API_KEY');
```
